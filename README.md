# 👗 Fashion Stylist AI ✨

Welcome to **Fashion Stylist AI** – your personal AI-powered fashion stylist! 💃🕺  
Upload your selfies & clothes, select the occasion and vibe, and get **perfect outfit suggestions** to shine in every moment. 🌟  

---

## 🎨 Features

- **👚 Personalized Styling:** Upload your clothes and selfies for AI-generated outfit combos.  
- **🎉 Occasion & Vibe Input:** Customize recommendations for parties, casual outings, dates, and more.  
- **📸 Multi-File Upload:** Upload multiple clothing items and see the best combinations.  
- **🖼️ Pinterest-Inspired UI:** Beautiful, clean, and responsive interface.  
- **⚡ Real-Time AI Suggestions:** Powered by **OpenAI GPT-4o**, analyzing text + images.  
- **👀 Preview Uploads:** See the clothes you’ve uploaded before AI creates suggestions.  

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript (Pinteresty UI vibes 🎨)  
- **Backend:** Node.js, Express  
- **AI:** OpenAI GPT-4o (text + image API)  
- **File Uploads:** Multer  
- **Optional Database:** Can store user profiles & history
